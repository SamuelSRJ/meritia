Sistema web para avalidar compatibilidade de curriculos com as vagas desejadas.

## Descrição

O sistema **Nome do sistema** é uma plataforma que recebe o curriculo e descrição da vaga desejada pelo usuário e faz uma analise atraves de Inteligencia Artificial **Google Gemini** avaliando os pontos fortes, pontos de melhoria, score de tech skills, soft skills e de compatibilidade com a vaga além de recomendações personalizadas para cada vaga.

## Como executar

### 1. Clone o repositório

```sh
git clone https://github.com/SamuelSRJ/xxxxxxxxxx.git
cd [Nome da aplicação]

yarn install
yarn dev
```

-Acesse o site do Google API Studio: https://aistudio.google.com/
-Efetue o login com sua conta Google ou cadastro na plataforma;
-No canto inferior esquerdo selecione “Get API Key”;
-Em seguida, no canto superior direito “Criar chave de API”;
-De um nome para essa chave e de preferencia crie o projeto que essa chave será utilizada;
-Copie a chave de API e insira no arquivo .env seguindo o formato de exemplo do arquivo .env.example

**OBS.:** O modelo de Inteligencia Artificial do Gemini que foi utilizado na construção dessa aplicação foi o Gemini 2.5 Flash que é **100% gratuito** e não necessita o cadastro de cartão de crédito. Caso você possua algum plano pago do Gemini verifique os modelos disponiveis e sinta-se livre para utiliza-lo.
